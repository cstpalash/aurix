## [1.0.0] - 2026-03-11

### Added
- **Confidence-Driven Autonomy** - Progressive trust system using Wilson score confidence intervals
- **AI-Powered Code Review** - Intent detection and semantic risk assessment using GPT models
- **Graduation System** - Four levels from Shadow to Full Auto based on demonstrated performance
- **GitHub Actions Integration** - Zero-cost infrastructure running on GitHub Actions
- **Configurable AI Models** - Support for gpt-4o-mini (default), gpt-4o, gpt-4-turbo
- **Risk Detection** - Automatic flagging of security, auth, database, and infrastructure changes
- **Auto-Demotion** - Automatic confidence adjustment on review disagreements
- **Audit Trail** - Full JSON-based logging for compliance and debugging

### Security
- No hardcoded credentials
- GitHub Secrets for API key management
- Minimal token permissions
